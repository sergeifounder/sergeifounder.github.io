Что такое GEO и зачем это нужно

SEO = попасть в результаты Google/Яндекс (синие ссылки)

GEO = быть процитированным AI-поисковиками (ChatGPT, Perplexity, Google AI Overviews, Яндекс Нейро)

SEO = тебя находят. GEO = тебя цитируют как эксперта.

Как продвигать Ross Painting Group

Текущий статус (из аудита geoseo.today)

SEO 86Хороший уровень. Основные проблемы: duplicate titles (16 групп), heading skips
GEO 94Отличный уровень. Schema есть, FAQ есть, E-E-A-T сигналы
127страниц. 3 региона: San Diego, Los Angeles, Orange County

Что нужно починить (приоритет)

  • P016 дублей titles — /services/X/ и /services/X-san-diego/ имеют одинаковые title. Уникализировать
  • P043 дубля descriptions — 43 location pages с одним meta description. Написать уникальные
  • P193 картинки без alt — все service pages. Добавить описательный alt text
  • P16 orphan pages — нет внутренних ссылок. Добавить в навигацию
  • P1Heading skip H1→H3 — на 80+ страницах. Добавить H2 между H1 и H3
  • P2Нет OG tags — добавить og:title, og:description, og:image для шеринга
  • P2Нет canonical — добавить canonical URL на все страницы
  • P2sameAs ссылки — добавить в Schema: Wikipedia, LinkedIn, BBB

GEO-стратегия для Ross Painting

  • 1YouTube канал — 3-5 видео: "Before/After painting", "How to choose painter", "Our process". YouTube = #1 сигнал для AI
  • 2Google Business — уже есть 17 отзывов. Цель: 50+ отзывов. Просить каждого клиента
  • 3Yelp — активный профиль, отвечать на все отзывы
  • 4Блог — 1 статья/месяц: "Cost of painting in La Jolla 2026", "Best exterior paint for San Diego climate"
  • 5Reddit r/SanDiego — отвечать на вопросы про painters (Perplexity цитирует Reddit)
  • 6FAQ расширить — добавить 10-15 вопросов с конкретными цифрами (AI любит факты)
  • 7Даты на страницах — добавить datePublished/dateModified. AI считает свежий контент надёжнее

Универсальный чеклист GEO для любого сайта

Техническая база (проверяет geoseo.today)

  • ✓robots.txt разрешает GPTBot, ClaudeBot, PerplexityBot
  • ✓Title tag 30-60 символов, уникальный на каждой странице
  • ✓Meta description 120-160 символов, уникальный
  • ✓H1 → H2 → H3 иерархия без пропусков
  • ✓Schema JSON-LD: Organization + LocalBusiness + FAQPage
  • ✓Canonical URL на каждой странице
  • ✓HTTPS + быстрая загрузка (LCP < 2.5 сек)
  • ✓Alt text на всех картинках
  • ✓Sitemap.xml в robots.txt

Контент для AI-цитирования

  • ✓BLUF — ответ в первом абзаце. Не "введение", а сразу суть
  • ✓Цифры — конкретные данные: "$3,500-$8,000", "15+ лет опыта", "500+ проектов"
  • ✓Списки — буллеты и нумерация. AI извлекает списки лучше чем прозу
  • ✓FAQ — вопрос (H2/H3) + ответ 2-4 предложения. Идеально для AI
  • ✓Автор — имя, должность, фото, соцсети. AI доверяет подписанному контенту
  • ✓Даты — datePublished + dateModified. Обновлять минимум раз в 6 месяцев
  • ✓Внешние ссылки — ссылаться на .gov, .edu, исследования. Princeton: +30% видимость
  • ✓Чанки 40-60 слов — самодостаточные абзацы которые AI может извлечь целиком

Бренд и присутствие

  • ✓YouTube — минимум 1 видео. Лучше 3-5. #1 сигнал для AI (r=0.737)
  • ✓Google Business Profile — полностью заполнен, 10+ отзывов
  • ✓Yelp / отраслевые каталоги — присутствие + ответы на отзывы
  • ✓LinkedIn компании — страница + регулярные посты
  • ✓sameAs в Schema — ссылки на все профили
  • ✓Название бренда консистентно — одинаковое в title, H1, OG, Schema, footer
  • ✓Бренд в первых 100 словах — AI считывает opening content для entity recognition

Яндекс Нейро (для РФ проектов)

Ключевое отличие: Яндекс Нейро берёт ответы ТОЛЬКО из Топ-5 органики Яндекса. Значит сначала нужен классический SEO, потом GEO.

  • 1Быть в Топ-5 Яндекса — без этого Нейро тебя не увидит
  • 2Ответ в первом экране — 2-4 предложения сразу после H1
  • 3Поведенческие факторы — Яндекс уникально важен: dwell time, глубина скролла
  • 4YATI алгоритм — раскрывай тему глубоко, не spamь ключевыми словами
  • 5E-E-A-T паспорт — автор с должностью, соцсетями, датами. Критично для Яндекса
